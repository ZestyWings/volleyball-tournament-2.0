Volleyball Tournament Manager

Vanilla HTML + JS. Team check-in, pool scheduling, scoring, standings, and a 10-seed elimination bracket (1 and 2 get byes). Pool rounds auto-match the number of teams: 8→4 rounds/4 games, 10→5 rounds/4 games + Bye row, 12→4 rounds/6 games, 14→4 rounds/7 games, 16→4 rounds/8 games (odd counts create a BYE match).
